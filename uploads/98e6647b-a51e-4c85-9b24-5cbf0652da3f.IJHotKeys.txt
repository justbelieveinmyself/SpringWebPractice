Ctrl + P - параметры метода
Ctrl + Q - документация к методу
Ctrl + B — переход в исходный код класса\метода либо переход к объявлению переменной
Alt + F7 — покажет, где используется переменная\метод
Shift + Shift (Double Shift, 2 раза подряд быстро нажать shift) — поиск всего и везде (ищет классы и файлы но не методы). 
Ctrl + Shift + V – умная вставка, помнит несколько последних скопированных элементов. 
Ctrl + Shift + Space – умный комплишен
Ctrl + D - дубликат строки
Alt + D - удалить строку
Ctrl + Shift + A – поиск хоткея
Ctrl + Alt + V – если вы написали, что-то и надо положить это в переменную
Ctrl + Alt + M - выделит кусок кода в метод
Alt + Enter – комплишен для решения любых проблем
Alt + Insert – автогенерация всего
Ctrl + O –переопределение методов родителя
Ctrl + Alt + S – настройки IDEA
Ctrl + Alt + Shit + S – настройки проекта
Ctrl + Shift + U - смена регистра у слова
Ctrl + ALT + L  - выравнивание кода, отступы
Ctrl + F12 - увидеть все методы и переменные класса
Ctrl + Shift + F9 - обновить html
